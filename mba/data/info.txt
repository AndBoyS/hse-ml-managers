Банк всегда выдает кредит под 10% годовых, процент начисляется ежемесячно

- цель - цель кредита
- тип - тип кредита
- тип кредита - из какого источника берутся данные по кредитному скору клиента
- бизнес - используется ли кредит для коммерческих целей
- работа - тип проживания (основное, вторичное, для инвестирования). Проживание в недвижимости, которую клиент покупает на кредит
- кредит нарушен - тут речь идет про conforming/non-corforming loans
- кредитоспособность - два типа кредитоспособности
- прямой залог - используется ли для обеспечения кредита прямой залог
- амортизация - наличие негативной амортизации
- только процент - тип ипотеки, где какое-то время платится только процент
- один платеж - выплата кредита производится одним платежом
